Gerar Conversa

Para usar esse gerador

  1. Selecione o pais ao qual pertence o número
  2. Informe o numero
  3. Digite uma mensagem (opcional)
  4. Clique em gerar

Importante lembrar que esse gerador não cria numeros fakes para conversas no WhatsApp.
